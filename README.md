# ShanzhaiTaiko-OpenSource
用UNITY3D制作的山寨太鼓达人游戏taiko no tatsujin

说明：

1、UNITY3D版本：2020.3.1

2、TJA文件分析代码抄袭自TJAPlayer3

3、涉及版权的图形素材和模型已删除或修改

4、模型描边取自https://github.com/Arvtesh/UnityFx.Outline
因模型删除在场景中该脚本已关闭避免报错

5、音效引擎使用CriWare
https://www.criware.com/

6、插件表
Newtonsoft.Json
SharpDX
SharpDX.DirectInput
